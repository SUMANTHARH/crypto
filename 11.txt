a.Demonstrate how to Send ICMP packets in Scapy and show the packet captured using Wireshark to destination 8.8.8.8.
Ans:Open cmd prompt(normal)
 Next,type pip install scapy  //(if not already installed)
 Then type: scapy //hit enter
 Type the following cmds below:
>>> frame=Ether()/IP(dst="8.8.8.8")/ICMP()
>>> frame //U will get this as output ..<Ether  type=IPv4 |<IP  frag=0 proto=icmp dst=8.8.8.8 |<ICMP  |>>>
After that open wireshark and start capture,and come back to cmd prompt and type
>>> sendp(frame) //capture the packet in wireshark...apply icmp filter and observe it
