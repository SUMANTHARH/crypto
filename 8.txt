a.Write a python code using Scapy for creating a Network Scanner (Hint: Create an ARP packet using ARP() method ).
Ans:Open cmd prompt(normal)
 Next,type pip install scapy  //(if not already installed)
 Then type: scapy //hit enter
 Type the following cmd below:
     
from scapy.all import *
request = ARP()
request.pdst = '192.168.6.0/24'
broadcast = Ether()
broadcast.dst = "ff:ff:ff:ff:ff:ff"
req_broadcast = broadcast / request
clients = srp(req_broadcast, timeout=3, verbose=False)[0]
if not clients:
    print("No clients found or no response received.")
else:
    for element in clients:
        print(element[1].psrc + " " + element[1].hwsrc) // u will get this output: 192.168.6.149 00:45:e2:c0:b1:fd 192.168.6.81 f6:31:8e:7d:ba:ab
